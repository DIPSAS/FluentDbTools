Resolve table-value by template and tableName
i.e: With template equal to "tables:{tableName}:globalId" and tableName equal to "Person" (and ie. migrationConfig.GetSchemaPrefixId() (GetSchemaPrefixId()) returns "EX"

  • "database:migration:tables:Person:globalId"
  • "database:migration:tables:EXPerson:globalId"
  • "database:tables:Person:globalId"
